# Contributing

## Rebuild source:

In case you want to build your own bundle for client

1. Clone this repository

    ```bash
    git clone git@github.com:ticketmaster-api/sdk-javascript.git
    ```

1. install dependencies

    ```bash
    npm install
    ```

1. Run npm script:

    ```bash
    npm run build
    ```

## Running Tests

 ```bash
 npm test
 ```
